Christian Adinata: Saya Tidak Marah, tetapi Sedih dan Hancur

Senin, 6 Januari 2025 07.30 WIB

Semarang - Christian Adinata adalah salah satu pemain yang sempat disebut memiliki prospek bagus di masa depan dan bisa jadi harapan akan hadirnya regenerasi di nomor tunggal putra. Namun setelah memenangkan SEA Games 2023 saat ia belum genap berusia 22 tahun, Christian Adinata mengalami cedera saat tampil di Malaysia Masters.

Awal cedera, Christian Adinata tidak langsung menjalani operasi melainkan hanya menjalani terapi. Setelah berbulan-bulan, Christian Adinata sempat tampil lagi di lapangan namun kondisinya tidak maksimal. Christian Adinata akhirnya menjalani operasi di pertengahan tahun 2024.

Saat menjalani pemulihan, Christian Adinata mendapati kabar bahwa dirinya terkena degradasi dari Pelatnas Cipayung. Berikut wawancara eksklusif CNNIndonesia.com dengan Christian Adinata: Bagaimana cerita awalnya saat keputusan dari PBSI itu muncul? Apakah kamu sudah tahu sebelum dirilis ke publik?

Keputusan itu, saya baru tahu dari Sabtu pagi. Jadi memang dari sebelumnya, saya belum tahu apa-apa. Waktu sebelum dipulangkan, ada meeting pelatih. Dari pelatih fisik, ada sedikit bocoran kala saya masih aman. Karena saya masih dalam pemulihan.

Jadi pikir saya, karena masih pemulihan, saya masih dikasih waktu oleh PBSI sampai benar-benar sembuh. Baru nanti dilihat lagi seperti apa. Jadi saya merasa aman. Tiga hari sebelum pengumuman, saya sempat telepon Kabid Binpres Koh Didi [Eng Hian]. Cerita sebelumnya, PBSI meminta medical report saya, jadi saya disuruh MRI, terus ada tes kekuatan, mengukur kondisi lutut, segala macam. Pokoknya, mau melihat sudah sampai sejauh mana perkembangannya.

Saya sudah lakukan itu semua dan hasilnya membaik semua. Jadi tinggal diteruskan sedikit lagi. Dokter prediksi bulan Maret saya sudah bisa berkompetisi lagi. Dan tiga hari sebelum pengumuman keluar, saya telepon Koh Didi. Tujuannya menyampaikan hasil MRI dan tes-tes untuk melihat sejauh mana.

Tujuan selanjutnya saya menelepon itu, saya meminta request status saya yang utama untuk turun ke pratama. Karena saya merasa tidak layak untuk kondisi sekarang yang belum sempurna di skuad utama. Karena khawatir mengganggu program yang lain.

Apalagi dengan pelatih baru yang belum benar-benar mengerti kondisi saya. Jadi saya request turun ke pratama. Karena saya merasa kurang layak dan di pertandingan pun ranking saya sudah jauh sekali, jadi harus mulai dari bawah lagi.

Otomatis, pertandingan bakal mengikuti jadwal pratama. Jadi lebih baik saya di pratama dulu dan kerja keras lagi sampai ada momen untuk masuk skuad utama. Tetapi jawaban yang buat saya kaget, Koh Didi bilang tunggu dulu ini bukan masalah utama atau pratama, masih ada rapat untuk kamu dipertahankan atau tidak, jadi tunggu kabar dulu.

Terus ya sudah dari situ tidak ada kabar sama sekali sampai pada akhirnya hari Sabtu saya bangun tidur, ada DM masuk dari teman ganda putra bilang,'Semangat Chris, tetap semangat'. Saya kaget, memang ada apa nih. Kemudian saya cek di media sosial PBSI belum ada daftar, tetapi kemudian sudah berseliweran di twitter. Saya kaget nama saya tidak ada.
